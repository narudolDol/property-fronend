# Frontend (React + Vite)

Frontend for the real-estate listing demo.

## Requirements

- Node.js (LTS recommended)
- pnpm

## Install

```bash
cd frontend
pnpm install
```

## Run project (development)

Start backend (separate terminal):

```bash
cd backend
pnpm start:dev
```

Start frontend:

```bash
cd frontend
pnpm dev
```

## API base URL (optional)

Default backend base URL used by the frontend:
- `http://localhost:3001/api/v1`

## Build / Preview

Build:

```bash
cd frontend
pnpm build
```

Preview:

```bash
cd frontend
pnpm preview
```
